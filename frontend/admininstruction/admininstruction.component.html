ADMIN INSTRUCTIONS

1.You can add a trainer
  To add a trainer, fill in his details along with his location & specialization

2.You can accept or delete a trainer
  To accept a trainer, go to view requests & accept.
  To delete a trainer, go to trainer details page & delete

2.You can modify diet chart
  To add diet chart,select the prefernce combination from drop down menu and select the component you want to update. Fill in the calories and diet data to update for the given combination.

3.You can delete diet chart
  You can delete a diet chart by selecting combination and updating the text-area as blank.

4.You can modify fitness chart
  To add fitness chart,select the prefernce combination from drop down menu and select the component you want to update. Fill in the fitness data to update for the given combination.

5.You can delete fitness chart
  You can delete a fitness chart by selecting combination and updating the text-area as blank.

6.You can view all the trainers
